## VS Code 의 폰트는 WSL 과 관계없다

  - VS Code 에서 WSL 을 이용하고 있더라도 VS Code 는 Windows 에 설치되어 있기 때문에 WSL 에 폰트를 설치할 필요가 없다.
  - Windows 에 새로운 폰트를 설치한다면 바로 사용할 수 있다.
