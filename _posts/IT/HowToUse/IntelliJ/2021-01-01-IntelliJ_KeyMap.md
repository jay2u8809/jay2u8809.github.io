> IntelliJ(인텔리J)를 사용하면서 유용한 단축키에 대해 정리해 본다

> ### IntelliJ 유용한 단축키     

___


#### 1. 코드 작성에 관한 단축키

&nbsp; 
1) 해당 라인 위치 변경 : **` Ctrl[Cmd] + Shift + 방향키 Up/Down `**      
&nbsp;&nbsp; 
해당 라인의 위치를 변경한다. 위/아래로 옮길수 있다.     
<br />

&nbsp; 
2) 해당 라인 삭제 : **` Ctrl[Cmd] + y `**     
&nbsp;&nbsp; 
해당 라인을 삭제한다.       
<br />

&nbsp; 
3) 구현 메소드로 이동 : **` Ctrl[Cmd] + Alt + b `**     
&nbsp;&nbsp; 
해당 메소드가 구현되어 있는 곳으로 이동한다.        

<br /><br />


#### 2. 검색에 관한 단축키

&nbsp; 
1) 파일 검색 : **` Shift X 2 `**      
&nbsp;&nbsp; 
전체 프로젝트/라이브러리에서 원하는 파일을 검색한다. (shift 버튼을 연속적으로 2번 누른다.)     
<br />

&nbsp; 
2) 단어 검색 : **` Ctrl + Shift + f `**      
&nbsp;&nbsp; 
전체 프로젝트/라이브러리를 대상으로 원하는 단어를 검색한다.     

<br /><br />
