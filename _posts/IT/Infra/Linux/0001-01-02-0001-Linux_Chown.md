> Linux에서 파일과 디렉토리에 대한 권한을 변경하기 위한 Chown 명령어 사용하기


### Linux Chown 사용자(User) 변경
**`CHOWN`** 사용자명 디렉토리(파일)
```
    chown jian aaa.txt
```
<br />

### Linux Chown 그룹(Group) 변경
**`CHOWN`** 그룹명 디렉토리(파일)
```
    chown :grp-jian aaa.txt
```
<br />

### Linux Chown 사용자(User)와 그룹(Group) 함께 변경
**`CHOWN`** 사용자명:그룹명 디렉토리(파일)
```
    chown jian:grp-jian aaa.txt
```
<br />
