> 자바스크립트의 함수Function에 대한 간단한 정리

### 함수의 파라미터
##### 1) Default Paramter <br />
&nbsp;-&nbsp; 함수의 파라미터에 데이터를 넣지 않아도 기본 설정되어 있는 값을 넣도록 한다.
```javascript

  // Default Parameter를 설정하지 않은 경우
  function showMessage (msg, content) {

    if(typeof content === 'undefined') {
      
      content = 'nothing';
    }

    console.log(`msg : ${msg}, content : ${content}`);
  }

  // Default Parameter를 설정한 경우
  function showMessage (msg, content = 'nothing') {

    console.log(`msg : ${msg}, content : ${content}`);
  }

```
<br/>

##### 2) Rest Paramter<br />
&nbsp;-&nbsp; 갯수가 정해지지 않은 복수의 파라미터 <br />
&nbsp;&nbsp; **`...`** 을 이용한다.
```javascript

  function printAllParameter (...args) {

    for (const arg of args) {

        console.log(arg);
    }

    args.forEach( (arg) => console.log(arg) );
  }

  printAllParameter('JavaScript', 'VueJS', 'TypeScript');
  printAllParameter('JavaScript');
  printAllParameter();

```
<br /><br />

### 함수의 정의
##### 1) 함수 표현식 Function Expressions
&nbsp;-&nbsp; 함수를 변수에 대입하여 사용하는 방법(자바스크립트 인터프리터에 의한 Hoisting이 일어나지 않는다.)
```javascript

  /**
   * 함수 선언식 Function Declarations : 일반적인 함수의 선언 
   * 함수의 선언보다 먼저 호출해도 실행이 된다.(자바스크립트 인터프리터에 의한 Hoisting)
   * 자바스크립트 인터프리터가 함수의 선언(정의)을 가장 위로 올려(Hoisting)버리기에 에러가 발생하지 않는다.
   */
  showMessage('test');  // Hoisting : error (X)
  function showMessage (msg, content = 'nothing') {

    console.log(`msg : ${msg}, content : ${content}`);
  }
  showMessage('test');


  /**
   * 함수 표현식 FunctionExpressions : 변수에 담는 형태
   * Hoisting이 되지 않기에 소소코드의 실행 순서에 따라 실행된다. 
   * 선언(정의)보다 먼저 실행할 경우 에러가 발생한다.
   */
  showMsg('test');  // Error (O)
  const showMsg = function (msg, content = 'nothing') { // 익명함수

    console.log(`msg : ${msg}, content : ${content}`);
  }
  showMsg('test');

  showMsg2('test');  // Error (O)
  const showMsg2 = function showMessage(msg, content = 'nothing') { 

    console.log(`msg : ${msg}, content : ${content}`);
  }
  showMsg2('test');
  
```

##### 2) Arrow Function
&nbsp;-&nbsp; **`=>`** 를 이용하여 함수를 정의하는 방법. 반드시 **`익명 함수로 사용`** 해야 한다.
```javascript

  // 함수 표현식 Function Expression
  const shwoMessage = function () {

    console.log('Hello');
  };

  // Arrow Function으로 변환
  const shwoMessage = () => console.log('Hello');

  // 파라미터가 있는 Arrow Function
  const shwoMessage = 
    (msg, content) => console.log(`${msg}  ${content}`);

  // 파라미터가 있는 Arrow Function : 함수의 내용이 긴 경우 {} 를 사용하고 반드시 return을 한다.
  const shwoMessage = (msg, content) => {

    console.log(`${msg}  ${content}`)
    return msg + typeName;
  };

```
<br />

##### 3) IIFE(Immediately Invoked Function Expression) <br />
&nbsp;-&nbsp; 함수의 **`선언과 동시에 실행`** 시키기 위한 방법
```javascript

  // 함수를 ()로 감싸고 끝에 ()를 붙인다.
  (function iifeTest() {

    console.log('IIFE Test');
  })();

```

### 기타
##### ・ TypeScript의 함수 정의
&nbsp;-&nbsp; 함수의 파라미터와 return의 **`타입을 반드시 지정`** 해주어야 한다. <br />
&nbsp;&nbsp; (자바스크립의 경우 return이 없을 경우에는 undefined 를 반환한다.)
```javascript

  /**
   * @param  : message 문자열
   * @return : 숫자형
   */
  function showMessage (message : string) : number {

    console.log(message);
    return 0;
  } 

```
