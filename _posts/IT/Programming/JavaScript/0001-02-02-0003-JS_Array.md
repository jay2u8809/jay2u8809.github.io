> 자바스크립트의 Array에 대한 정리 

### Array
##### 1) Array의 선언
```javascript

  const arr1 = new Array();
  const arr2 = [1, 2, 3, 4, 5];

```
<br/>

##### 2) Array의 인덱스 : Array Index Position
```javascript

  const bookeGenre = ['novel', 'technology', 'poem'];
  
  console.log(bookeGenre);        
  // [Result] [ 'novel', 'technology', 'poem' ]
  console.log(bookeGenre.length);
  // [Result] 3
  console.log(bookeGenre[0]);
  // [Result] novel
  console.log(bookeGenre[1]);
  // [Result] technology    
  console.log(bookeGenre[2]);
  // [Result] poem
  
  console.log(bookeGenre[bookeGenre.length-1]);
  // [Result] poem : last index

```
<br />

##### 3) Array의 반복문 : Looping over an array
```javascript

  const bookeGenre = ['novel', 'technology', 'poem'];
  
  // 일반 Loop
  for(let i = 0; i < bookeGenre.length; i++) {
      console.log(bookeGenre[i]);
  }
  
  // for ... of (Array)
  for(let value of bookeGenre) {
      console.log(value);
  }
  
  // [Result] 
  //     novel
  //     technology
  //     poem
  
  
  // forEach
  bookeGenre.forEach(function(value, index) {
  
      console.log(`value : ${value}, index : ${index}`);
  });
  
  // forEach Arrow Function
  bookeGenre.forEach((value, index) => 
  
      console.log(`value : ${value}, index : ${index}`)
  );

  // [Result] 
  //     value : novel, index : 0
  //     value : technology, index : 1
  //     value : poem, index : 2

```
<br />

##### 4) Array의 추가, 삭제, 복사 : Addition, Deletion, Copy
&nbsp;-&nbsp; **`push()`** , **`pop()`**, **`unshift()`** , **`shift()`** , **`splice()`**, **`concat()`** <br />
&nbsp;-&nbsp; **`unshift()`** , **`shift()`** 는 push() 와 pop()에 비해 속도가 느리다. <br />
&nbsp;&nbsp; Array의 전체 인덱스를 다시 계산하는 처리가 추가되기 때문.
```javascript

  const bookeGenre = ['novel', 'technology', 'poem'];
  
  /**
   * push('요소') : 마지막 요소 추가, add an item to the end
   * push(...items: T[]): number;
   */
  bookeGenre.push('history', 'essay');
  console.log(bookeGenre);
  // [Result] : [ 'novel', 'technology', 'poem', 'history', 'essay' ]
  
  
  /**
   * pop() : 마지막 요소 삭제, remove an item from the end
   * pop(): T | undefined;
   */
  bookeGenre.pop();
  console.log(bookeGenre);
  // [Result] : [ 'novel', 'technology', 'poem', 'history' ]
   
  bookeGenre.pop();
  console.log(bookeGenre);
  // [Result] : [ 'novel', 'technology', 'poem' ]
  
  
  /**
   * unshift('요소') : 첫번째 요소 추가, add an item to the beginning
   * unshift(...items: T[]): number;
   */
  bookeGenre.unshift('history', 'essay', 'comic');
  console.log(bookeGenre);
  // [Result] : [ 'history', 'essay', 'comic', 'novel', 'technology', 'poem' ]
  
  
  /**
    * shift() : 첫 번째 요소 삭제, remove an item from the beginning
    * shift(): T | undefined;
    */
  bookeGenre.shift();
  console.log(bookeGenre);
  // [Result] : [ 'essay', 'comic', 'novel', 'technology', 'poem' ]

  bookeGenre.shift();
  console.log(bookeGenre);
  // [Result] : [ 'comic', 'novel', 'technology', 'poem' ]
  
  
  /** 
   * slice('시작 인덱스', '갯수') - 인덱스로 요소 삭제 remove an item by index position
   * splice(start: number, deleteCount?: number): T[];
   * splice(start: number, deleteCount: number, ...items: T[]): T[];
   */ 
  // 1번 인덱스부터 마지막 인덱스 요소 모두 삭제
  bookeGenre.splice(1);
  console.log(bookeGenre);
  // [Result] : [ 'novel' ]

  // 1번 인덱스부터 1개 요소 삭제 (1번만 삭제)  
  bookeGenre.splice(1, 1); 
  console.log(bookeGenre);
  // [Result] : [ 'novel', 'poem' ]
   
  // 1번 인덱스 요소 삭제 후 1번 인덱스에 요소 추가  
  bookeGenre.splice(1, 1, 'essay', 'history'); 
  console.log(bookeGenre);
  // [Result] : [ 'novel', 'essay', 'history', 'poem' ]
  
  
  /** 
   * concat('추가할 요소') - Array 뒤에 Array 추가
   * concat(...items: ConcatArray<T>[]): T[];
   */ 
   const bookAuthor = ['현진건', '심훈'];
   const newBookInfo = bookeGenre.concat(bookAuthor);
   console.log(newBookInfo);
   // [Result] : [ 'novel', 'technology', 'poem', '현진건', '심훈' ]

```

##### 5) Array 요소 검색 : Searching
&nbsp;-&nbsp; **`indexOf()`** , **`lastIndexOf()`**, **`includes()`** <br />
```javascript

  const bookeGenre = ['novel', 'technology', 'poem', 'technology'];
  
  /**
   * Find the index
   * indexOf('검색하고 싶은 값') : 해당하는 인덱스 값 반환 (앞에서부터)  
   * indexOf(searchElement: T, fromIndex?: number): number;
   */
   console.log(bookeGenre.indexOf('novel'));
   // [Result] : 0
   
   console.log(bookeGenre.indexOf('technology'));
   // [Result] : 1 (Array의 앞쪽 technology의 인덱스값)
   
   console.log(bookeGenre.indexOf('history'));
   // [Result] : -1 (검색값이 Array에 없을 경우)
   
  
  /**
   * lastIndexOf('검색하고 싶은 값') : 해당하는 인덱스 값 반환 (뒤에서부터)
   * lastIndexOf(searchElement: T, fromIndex?: number): number;
   */
   console.log(bookeGenre.lastIndexOf('novel'));
   // [Result] : 0
   
   console.log(bookeGenre.lastIndexOf('technology'));
   // [Result] : 3 (Array의 뒤쪽 technology의 인덱스값)
   
   console.log(bookeGenre.lastIndexOf('history'));
   // [Result] : -1 (검색값이 Array에 없을 경우)
   
  /**
   * includes('검색하고 싶은 값') : Array에 존재 여부를 Boolean으로 반환
   * includes(searchElement: T, fromIndex?: number): boolean;
   */  
   console.log(bookeGenre.includes('novel'));
   // [Result] : true
   
   console.log(bookeGenre.includes('technology'));
   // [Result] : true
   
   console.log(bookeGenre.includes('history'));
   // [Result] : false    
   
```
