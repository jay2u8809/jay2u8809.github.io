> 자바스크립트의 JSON에 대한 정리 

### JSON
##### 1) JSON이란?
&nbsp;-&nbsp; JSON : JavaScript Object Notation <br />
&nbsp;-&nbsp; **`데이터 통신을 위한 하나의 형식Format`** 으로서 key-value의 집합체인 자바스크립 객체 형태를 이용한 것. <br />
&nbsp;&nbsp;&nbsp;&nbsp; 1) 기존의 XML을 이용한 방식보다 가독성이 좋고 가벼워서 많이 사용된다. <br />
&nbsp;&nbsp;&nbsp;&nbsp; 2) 프로그래밍 언어와 플랫폼에 구분없이 사용할 수 있다. <br />
&nbsp;&nbsp;&nbsp;&nbsp; 3) 자바스크립트에만 존재하는 특별한 데이터나(Symbol 등) 함수Function 등은 포함되지 않는다.<br />
&nbsp;&nbsp;&nbsp;&nbsp; 4) JSON은 key값이나 문자열은 항상 쌍따옴표를 이용하여 표기해야한다. <br />
&nbsp;&nbsp;&nbsp;&nbsp; 5) JSON은 null, number, string, array, object, boolean을 사용할 수 있다. <br />
&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp; ※JSON은 **`데이터 형식Format일 뿐`** 이며 통신 방법도, 특정 프로그래밍 문법도 아닌 **`데이터를 표시하는 표현 방법`** 이다.
<br/>

##### 2) JSON으로 변환하기 Object to JSON
&nbsp;-&nbsp; **`JSON.stringify(데이터)`** <br /> 
&nbsp;-&nbsp; stringify(value: any, replacer?: (this: any, key: string, value: any) => any, space?: string | number): string; <br />
&nbsp;-&nbsp; stringify(value: any, replacer?: (number | string)[] | null, space?: string | number): string;
```javascript
    
    // Boolean
    let jsonObj = JSON.stringify(true);
    console.log(jsonObj);
    // [Result] : true
    
    
    // Array 
    jsonObj = JSON.stringify(['novel', 'poem']);
    console.log(json);
    // [Result] : ["novel","poem"]  ※double quotes로 변환된다
    
    
    // Object
    const book = {
        title :'Basic of JS',
        author : 'Jay Ian',
        price : null,
        publicationDate : new Date(),
        // symbol : Symbol('id'),   // 불가
        showBookInfo : () => {
            console.log(`Title : ${title}, Author : ${author}, Publication Date : ${publicationDate}`)
        },  
    }
    jsonObj = JSON.stringify(book);
    console.log(jsonObj);
    // [Result] : 함수(showBookInfo())는 데이터가 아니므로 포함되지 않았다.
    //      {
    //          "title": "Basic of JS",
    //          "author": "Jay Ian",
    //          "price": null,
    //          "publicationDate": "2020-08-16T05:43:59.160Z"
    //      }
    
    jsonObj = JSON.stringify(book, ['title']);
    console.log(jsonObj);
    // [Result] : 원하는 속성만 골라서 JSON으로 변환할 수도 있다. 
    //      {"title":"Basic of JS"}
    
    jsonObj = JSON.stringify(book, ['title', 'price']);
    console.log(jsonObj);
    // [Result]
    //      {
    //          "title": "Basic of JS",
    //          "price": null
    //      }
    
    // 함수를 이용해서 제어하며 변환할 수도 있다.
    jsonObj = JSON.stringify(book, (key, value) => {
        
        console.log(`key : ${key}, value : ${value}`);
        
        // author는 'Jay Ian' ⇒ 'Ian'으로 변경하여 변환 
        return key === 'author' ? 'Ian' : value;
    });
    console.log(jsonObj);
    // [Result]
    //      {
    //          "title": "Basic of JS",
    //          "author": "Ian",
    //          "price": null,
    //          "publicationDate": "2020-08-16T05:52:23.647Z"
    //       }
  
```

##### 3) JSON을 (자바스크립트 데이터로) 변환하기 JSON to Object
&nbsp;-&nbsp; **`JSON.parse(JSON)`** <br /> 
&nbsp;&nbsp; parse(text: string, reviver?: (this: any, key: string, value: any) => any): any;
```javascript

    // 위의 2) 코드에 이어서.
    
    // JSON ⇒ OBJECT
    const obj = JSON.parse(jsonObj);
    console.log(obj);
    // [Result] : 함수(showBookInfo())는 JSON으로 변환할 떄 제외되었기 때문에 존재하지 않는다.
    //    {
    //        title: 'Basic of JS',
    //        author: 'Ian',
    //        price: null,
    //        publicationDate: '2020-08-16T06:17:34.787Z'
    //     }
    book.showBookInfo();
    obj.showBookInfo(); // Error : showBookInfo()가 존재하지 않는다.
    
    // 함수를 이용해서 제어하며 변환할 수도 있다.
    obj = JSON.parse(jsonObj, (key, value) => {
    
        console.log(`key : ${key}, value : ${value}`);
        
        // publicationDate는 Date() 객체를 통해 변환하도록 한다.
        // 그냥 value를 반환하여 변환할 경우 Date()타입이 아니어서 getDate() 호출 시 에러가 발생한다.
        return key === 'publicationDate' ? new Date(value) : value;
        // return value;        // publicationDate.getDate() 호출 시 에러 발생
    });

    console.log(book.publicationDate.getDate());
    // [Result] : 16 (number Type)
    console.log(obj.publicationDate);
    // [Result] : 2020-08-16T06:24:23.512Z (object Type)
    console.log(obj.publicationDate.getDate());
    // [Result] : 16 (number Type)

```
