> 자바스크립트의 Object(객체)에 대한 간단한 정리 

### Object
##### 1) Object의 선언
&nbsp;-&nbsp; Object는 key-value의 집합체이다.
```javascript

  // 1. Object Literal Syntax
  const obj1 = {};
  
  // 2. Object Constructor Syntax
  const obj2 = new Object();

  // book Object
  const book = {   
    title  : 'Basic of JS',
    author : 'JayIan'
  };
  function showObjectValue (book) {
      console.log(book.title);
      console.log(book.author);
  }
  showObjectValue (book);


  /**
   * JS는 Dynamically Typed Language이므로 선언 이후에도 property를 추가하고 삭제할 수 있다.
   */ 

  // add genre property
  book.genre = 'Technology'; 
  console.log(book.genre);    // [Result] Technology

  // remove genre property
  delete book.genre;
  console.log(book.genre);    // [Result] undefined

```
<br/>

##### 2) Object의 Properties에 접근하는 방법
&nbsp;-&nbsp; Object의 Property는 key를 이용하여 value를 제어한다. <br />
&nbsp;&nbsp; **`key`** 는 반드시 **`문자열`** 로 활용한다.

```javascript

  // book Object
  const book = {   
    title  : 'Basic of JS',
    author : 'JayIan'
  };

  // title property 접근
  console.log(book.title);      // [Result] Basic of JS
  // console.log(book[title]); // error
  console.log(book['title']);   // [Result] Basic of JS

  // 장르genre 프로퍼티property 추가
  book['genre'] = 'Technology';
  console.log(book.genre);      // [Result] Technology
  console.log(book['genre']);   // [Result] Technology

  function printValue (obj, key) {
      console.log(obj[key]);
  } 
  printValue(book, 'author');   // JayIan

```
<br />

##### 3) Object 생성
&nbsp;-&nbsp; Object를 생성하기 위한 함수 <br />
&nbsp;&nbsp; 클래스를 선언하지 않고도 Object를 생성할 수 있다.
```javascript

  // 1. Property에 해당하는 key와 value를 그대로 넣어준다.
  const book1 = {   
    title  : 'Basic of JS',
    author : 'JayIan'
  };

  // 2-1. 함수를 통해 Object를 생성한다.
  const book21 = makeBook('Basic of JAVA', 'JayIan');
  console.log(book21);
  // [Result] { name: 'Basic of JAVA', age: 'JayIan' }
  function makeBook(title, author) {

    return {
        title,    // title : title,
        author,   // author : author,
    };
  }

  // 2-2. 함수를 통해 Object를 생성한다. (Constructor Function : 대문자와 new 키워드 사용)
  const book22 = new Booke('Basic of JAVA', 'JayIan');
  console.log(book22);
  // [Result] Book {title: "Basic of JAVA", author: "JayIan"}
  function Book (title, author) {

    // this = {};
    this.title = title;
    this.author = author;
    // return this;
  }

```

##### 4) In Operator (key in Obejct)
&nbsp;-&nbsp; Object내에 **`특정한 key의 존재 여부`** 를 판단 (Boolean) <br />
&nbsp;&nbsp; **`'key' in object`**
```javascript

  const book = {   
    title  : 'Basic of JS',
    author : 'JayIan'
  };

  console.log('title' in book);    // [Result] true
  console.log('author' in book);   // [Result] true
  console.log('genre' in book);    // [Result] false
  
```
<br />

##### 4) 반복문 Loop
&nbsp;-&nbsp; Object내의 key를 이용를 이용한 반복문 <br />
&nbsp;&nbsp; **`for 변수 in objcet`** (node.js의 경우 변수 부분에 let/const를 선언해야 한다.)
```javascript

  const book = {   
    title  : 'Basic of JS',
    author : 'JayIan'
  };

  // for 변수 in object
  for(key in book) {  // node.js : let key in book / const key in book
    console.log(`key : ${key}, value : ${book[key]}`);
  }
  // [Result]
  //    key : title, value : Basic of JS
  //    key : author, value : JayIan
  
```
<br />

&nbsp;-&nbsp; 순서가 있는(Iterable) 자료구조(Data Type)의 반복문 Loop <br />
&nbsp;&nbsp; **`for 변수 of 데이터`** (node.js의 경우 변수 부분에 let/const를 선언해야 한다.)
```javascript

  const bookList = ['Basic of JS', 'Advanced JS', 'JS Programming'];

  // for 변수 of 데이터
  for(value of bookList) {  // node.js : let value of bookList / const value of bookList
    console.log(`book : ${value}`);
  }
  // [Result]
  //    book : Basic of JS
  //    book : Advanced JS
  //    book : JS Programming
  
```
<br />

##### 5) Object의 복사(Cloning)
&nbsp;-&nbsp; Object를 복사하는 여러가지 방법 <br />
&nbsp;&nbsp; **`Object.assign(dest, [object1, object2, object3 ...])`**; <br />
&nbsp;&nbsp; Object.assign(복사할 객체, [복사할 객체1, 복사할 객체2, 복사할 객체3 ...]); <br />
&nbsp;&nbsp; (assign<T, U>(target: T, source: U): T & U;)
```javascript

  const book = {   
    title  : 'Basic of JS',
    author : 'JayIan'
  };

/**
 * 얉은 복사
 * 얉은 복사는 포인터(메모리 주소)가 동일하기 때문에
 * 원본 또는 사본의 데이터를 변경하면 모두의 데이터가 변경된다.
 */

  // book을 book2에 대입 (얉은 복사)
  const book2 = book;

  // book2의 제목을 변경
  book2.title = 'Basic of Java';

  console.log(book.title);     
  // [Result] Basic of Java
  console.log(book2.title);
  // [Result] Basic of Java


/**
 * 깊은 복사
 * 깊은 복사는 포이터(메모리 주소)가 다르기 때문에
 * 원본과 사본이 별개이다.
 */
  
  /** 1) Old Way  **/
  const book3 = {};         // 빈 객체를 생성
  for(key in book) {
    book3[key] = book[key]; // 각 Property를 복사
  }
  
  console.log(book.title);
  // [Result] Basic of JS
  console.log(book3.title);
  // [Result] Basic of JS

  // book3의 제목을 변경
  book3.title = 'JS Programming';

  console.log(book.title);
  // [Result] Basic of JS
  console.log(book3.title);
  // [Result] JS Programming


  /** 2) New Way : Object.assign(dest, [object1, object2, object3 ...]) 사용 **/
  const book4 = {};         // 빈 객체를 생성
  
  // 객체 복사
  Object.assign(book4, book);
  console.log(book4);
  // [Result] { title: 'Basic of JS', author: 'JayIan' }

  const book5 = Object.assign({}, book);
  console.log(book5);
  // [Result] { title: 'Basic of JS', author: 'JayIan' }


  // 복사할 객체의 순서에 따른 복사 객체의 변화
  const novel1 = {  title : '토지'  };
  const novel2 = {  title : '운수 좋은 날', author : '현진건'  };
  const novel3 = {  title : '상록수', author : '심훈'  };
  
  const novelMixed12 = Object.assign({}, novel1, novel2);
  console.log(novelMixed12);
  // [Result] { title: '운수 좋은 날', author: '현진건' }

  const novelMixed21 = Object.assign({}, novel2, novel1);
  console.log(novelMixed21);
  // [Result] { title: '토지', author: '현진건' }

  const novelMixed123 = Object.assign({}, novel1, novel2, novel3);
  console.log(novelMixed123);
  // [Result] { title: '상록수', author: '심훈' }

  const novelMixed231 = Object.assign({}, novel2, novel3, novel1);
  console.log(novelMixed231);
  // [Result] { title: '토지', author: '심훈' }

  const novelMixed321 = Object.assign({}, novel3, novel2, novel1);
  console.log(novelMixed321);
  // [Result] { title: '토지', author: '현진건' }
  
```
