> 자바스크립트의 Array API에 대한 정리 

### Array API
##### 1) JOIN()
&nbsp;-&nbsp; **`join(구분자)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; join(separator?: string): string; <br />
&nbsp;&nbsp;&nbsp;&nbsp; **`배열 ⇒ 문자열`** 배열의 요소들을 하나의 string으로 만들어 반환
```javascript
    
    const bookGenre = ['history', 'art', 'novel'];
    
    const bookGenreStr = bookGenre.join('|');
    console.log(bookGenreStr);
    // [Result] : history|art|novel (Type : string)
    
    const bookGenreStr2 = bookGenre.join(', * ');
    console.log(bookGenreStr2);
    // [Result] : history, * art, * novel (Type : string)
    
    // join()의 파라미터가 없을 경우, 자동으로 ','(콤마)가 들어간다. 
    const bookGenreStr3 = bookGenre.join();
    console.log(bookGenreStr3);
    // [Result] : history,art,novel (Type : string)

```
<br/>

##### 2) SPLIT()
&nbsp;-&nbsp; **`split(구분자)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; split(separator: string | RegExp, limit?: number): string[]; <br />
&nbsp;&nbsp;&nbsp;&nbsp; **`문자열 ⇒ 배열`** string을 파라미터 기준으로 잘라 배열을 만든다.
```javascript
    
    const bookGenre =  'history, art, novel';
    
    // 파라미터(','(콤마)) 기준으로 string을 잘라 배열을 생성한다.
    const bookGenreStr = bookGenre.split(',');
    console.log(bookGenreStr);
    // [Result] : [ 'history', ' art', ' novel' ] (length : 3)
    
    
    // 파라미터(','(콤마)) 기준으로 string을 잘라 앞의 2개만 요소로만 배열을 생성
    const bookGenreStr2 = bookGenre.split(',', 2);
    console.log(bookGenreStr2);
    // [Result] : [ 'history', ' art' ] (length : 2)
    
    // split()의 파라미터가 없을 경우, 그대로 길이가 1인 배열의 요소가 된다.
    const bookGenreStr3 = bookGenre.split();
    console.log(bookGenreStr3);
    // [Result] : [ 'history, art, novel' ] (length : 1)

```
<br/>

##### 3) REVERSE()
&nbsp;-&nbsp; **`reverse()`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; reverse(): T[]; <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열 요소들의 순서를 **`역순`**으로 만든다.
```javascript
    
    const orginalArr = [1, 2, 3, 4, 5];
    const reverseArr = orginalArr.reverse();

    // 원본 배열과 역순으로 생성된 배열 모두 순서가 변경된다.
    console.log(orginalArr);     // [Result] : [ 5, 4, 3, 2, 1 ]
    console.log(reverseArr);     // [Result] : [ 5, 4, 3, 2, 1 ]

```
<br/>

##### 4) SLICE()
&nbsp;-&nbsp; **`slice(시작인덱스, 마지막인덱스 + 1)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; slice(start?: number, end?: number): T[]; <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열의 일부를 새로운 배열로 만든다.
```javascript
    
    const orginalArr = [1, 2, 3, 4, 5];
    
    const sliceArr = orginalArr.slice(2, 4);       // index:2 ~ index:3까지
    console.log(orginalArr);     // [Result] : [ 1, 2, 3, 4, 5 ]
    console.log(sliceArr);       // [Result] : [ 3, 4 ]
    
    // ※ splice()와의 차이점
    const spliceArr = orginalArr.splice(2, 2);     // index:2 ~ index:3까지
    console.log(orginalArr);     // [Result] : [ 1, 2, 5 ] (원본이 수정됨)
    console.log(spliceArr);      // [Result] : [ 3, 4 ]
    
```
<br/>

##### 5) FIND()
&nbsp;-&nbsp; **`find(콜백함수)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열 요소들 중 조건이 true인 요소가 있는지 확인 <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열의 각 요소들마다 콜백함수가 실행되어 true 가 반환되면 실행을 멈춘다.
```javascript
    
    class Book {
        constructor(title, author, rate, rePublishing) {
            this.title = title;
            this.author = author;
            this.rate = rate;
            this.rePublishing = rePublishing;
        }
    }
    
    const books = [
        new Book('Basic of JS', 'Ian', 1.4, false),
        new Book('Basic of HTML', 'Bran', 3.3, true),
        new Book('Basic of Java', 'John', 4.1, true),
        new Book('Basic of C#', 'Tom', 2.6, true),
        new Book('Basic of Kotlin', 'Simon', 2.1, false),
        new Book('Basic of CSS', 'Jay', 4.5, true),
    ];
    
    // books의 요소들 중 평점이 4.0 이상인 배열이 존재한다면 반환하고 종료
    const result = books.find( (book, index) => {
        return book.rate > 4.0;
    });

    const result2 = books.find( (book) => 
        book.rate > 4.0
    );

    console.log(result);
    console.log(result2);
    // [Result] : result와 result2는 동일한 결과를 반환
    //    Book {
    //        title: 'Basic of Java',
    //        author: 'John',
    //        rate: 4.1,
    //        rePublishing: true
    //    }
    
```

##### 6) FILTER()
&nbsp;-&nbsp; **`filter(콜백함수)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열 요소들 중 조건이 true인 요소들을 Array로 생성하여 반환한다. <br />
```javascript
       
    // books의 요소들 중 재출판이 아닌 요소(book.rePublishing === false)만 Filter한다. 
    const republishingBooks = books.filter( (book) =>
     
        // book.rePublishing === false 로 표현해도 가능
        !book.rePublishing
    );

    console.log(republishingBooks);
    // [Result]
    //    [
    //        Book {
    //            title: 'Basic of JS',
    //            author: 'Ian',
    //            rate: 1.4,
    //            rePublishing: false
    //        },
    //        Book {
    //            title: 'Basic of Kotlin',
    //            author: 'Simon',
    //            rate: 2.1,
    //            rePublishing: false
    //        }
    //    ]
    
```
<br/>

##### 7) MAP()
&nbsp;-&nbsp; **`map(콜백함수)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; map<U>(callbackfn: (value: T, index: number, array: T[]) => U, thisArg?: any): U[]; <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열 요소들을 **`다른 데이터 타입으로 변환Mapping`** 하여 새로운 배열을 생성 
```javascript
       
    // 책 제목만 존재하는 배열을 새로 생성
    const titleMappingArr = books.map( (book) =>
     
        book.title
    );

    console.log(titleMappingArr);
    // [Result]
    // [
    //   'Basic of JS',
    //   'Basic of HTML',
    //   'Basic of Java',
    //   'Basic of C#',
    //   'Basic of Kotlin',
    //   'Basic of CSS'
    // ]
    
    
    // 책 평점만 존재하는 배열을 새로 생성
    const rateMappingArr = books.map( (book) =>
     
        book.rate
    );

    console.log(rateMappingArr);
    // [Result] : [ 1.4, 3.3, 4.1, 2.6, 2.1, 4.5 ]
    
```
<br/>

##### 8) SOME() 과 EVERY()
&nbsp;-&nbsp; **`some(콜백함수)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; some(callbackfn: (value: T, index: number, array: T[]) => unknown, thisArg?: any): boolean; <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열의 모든 요소들 중 **`조건에 부합하는 요소가 1개라도 존재한다`** 면 true를 반환 <br />
&nbsp;-&nbsp; **`every(콜백함수)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; every(callbackfn: (value: T, index: number, array: T[]) => unknown, thisArg?: any): boolean; <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열의 모든 요소들이 **`모두 조건에 부합한다`** 면 true를 반환 
```javascript
       
    // books의 요소들 중 평점이 2.5이하인 책이 하나라도 존재하는가?
    const isExist1 = books.some( (book) => book.rate <= 2.5);
    console.log(isExist1);
    // [Result] : true
    
    // books의 요소들 중 평점이 4.5 초과인 책이 하나라도 존재하는가?
    const isExist2 = books.some( (book) => book.rate > 4.5);
    console.log(isExist2);
    // [Result] : false


    // books의 요소들이 모두 평점 3.0 이상인가?
    const isExist3 = books.every( (book) => book.rate > 3.0);
    console.log(isExist3);
    // [Result] : false
    
    // books의 요소들이 모두 평점 1.0 이상인가?
    const isExist4 = books.every( (book) => book.rate > 1.0);
    console.log(isExist4);
    // [Result] : true
    
```
<br/>


##### 9) REDUCE()
&nbsp;-&nbsp; **`reduce(콜백함수, 초기값)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열의 모든 요소들을 대상으로 누적하여 처리
```javascript

    // 각 책들의 평점의 합계를 구한다.
    const rateSum = books.reduce( (prev, current) => {
        return prev + current.rate;
    }, 0);
    console.log(rateSum);
    // [Result] : 18 (0 + books[0].rate + book[1].rate + ... + book[5].rate)


    rateSum = books.reduce( (prev, current) => prev + current.rate, 0);
    // [Result] : 18 
```
<br/>

##### 9) SORT()
&nbsp;-&nbsp; **`sort(콜백함수)`** <br />
&nbsp;&nbsp;&nbsp;&nbsp; sort(compareFn?: (a: T, b: T) => number): this; <br />
&nbsp;&nbsp;&nbsp;&nbsp; 배열 요소를 정렬한다.
```javascript

    // 각 책들을 평점이 낮은 순(오름차순)으로 정렬한다.
    const sortedAscBooks = books.sort( (a, b) => a.rate - b.rate);
    console.log(sortedAscBooks);
    console.log(books);
    // [Result] : books 원본의 정렬이 변경되어 있음을 확인할 수 있다.

    // 각 책들을 평점이 높은 순(내림차순)으로 정렬한다.
    const sortedDescBooks = books.sort( (a, b) => b.rate - a.rate);
    console.log(sortedDescBooks);
    console.log(books);
    // [Result] : books 원본의 정렬이 변경되어 있음을 확인할 수 있다.
    
```

##### 10) Method Chaining
&nbsp;-&nbsp; 위의 메소드들을 이어서 사용할 수 있다.<br />
```javascript

    // 책의 작가 이름들 문자열 생성
    const result1 = books
        .map( (book) => book.author)
        .join(); 
    console.log(result1);
    // [Result] : Ian,Bran,John,Tom,Simon,Jay

    // 책의 평점이 2.5 초과인 책들의 평점들 문자열 생성
    const result2 = books
        .map( (book) => book.rate)
        .filter( (rate) => rate > 2.5)
        .join(); 
    console.log(result2);
    // [Result] : 3.3,4.1,2.6,4.5

    // 책의 평점들을 내림차순으로 정렬하여 문자열 생성
    const result3 = books
        .map( (book) => book.rate)
        .sort( (a, b) => b - a)     // desc
    .join();
    console.log(result3);
    // [Result] : 4.5,4.1,3.3,2.6,2.1,1.4

```
