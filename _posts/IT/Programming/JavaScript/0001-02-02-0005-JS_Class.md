> 자바스크립트의 클래스Class에 대한 간단한 정리 

### 클래스Class
##### 1) 클래스의 정의 Class Declaration
&nbsp;-&nbsp; 생성자와 클래스 내부의 메소드를 정의한다.
```javascript

  class Book {

    // 생성자Contructor
    constructor(title, author) {
        // initial field
        this.title = title;
        this.author = author;
    }
    
    // method
    showInfo() {
        console.log(`Book Title : ${this.title}, Author : ${this.author}`);
    }
  }

  const book = new Book('Basic of JS', 'BranJay');
  console.log(book.title);
  // [Result] Basic of JS
  console.log(book.author);
  // [Result] BranJay
  console.log(book.showInfo());
  // [Result] Book Title : Basic of JS, Author : BranJay

```
<br/>

##### 2) Getter and Setter 메소드
&nbsp;-&nbsp; 클래스 내부의 필드Field에 접근하고 값을 변경할 **`Getter와 Setter`**  메소드를 정의
```javascript

  class Book {

    // 생성자Contructor
    constructor(title, author) {
      // Initial field
      this.title = title;
      this.author = author;
    }

    // method
    showInfo() {
      console.log(`Book Title : ${this.title}, Author : ${this.author}`);
    }
  
    // Getter
    get title() {
      return this._title;
      // return this.title을 사용할 경우 Getter 메소드가 계속 호출된다.
      // 따라서 _title이라는 새로운 필드를 만들어 title이라는 변수의 값을 반환하도록 한다.
    }

    // Setter
    set title(value) {
      if(value === null || value === undefined) {
        throw Error('');
      }
      this._title = value;
    }

  }
  
  const book1 = new Book('Basic of JS', 'Jay Ian');
  console.log(book1.title);
  // [Result] Basic of JS
  console.log(book1.author);
  // [Result] Jay Ian
  console.log(book1._title);
  // [Result] Basic of JS
  console.log(book1._author);
  // [Result] undefined

```
<br />

##### 3) 접근 제어자 : public과 private
&nbsp;-&nbsp; **`캡슐화`** 를 위한 접근 제어자
```javascript

  class Book {

    title = 'Basic of JS';  // public 
    #author = 'Jay Ian';    // private 
  }
  
  const book1 = new Book();
  console.log(book1.title);
  // [Result] Basic of JS
  console.log(book1.author);
  // [Result] undefined
  
  // private의 경우 아직 최신의 브라우저에서도 사용할 수 없는 경우가 있다.

```
<br />

##### 4) Static
&nbsp;-&nbsp; 다른 객체지향언어들과 마찬가지로 **`Static`** 은 클래스의 변수, 메소드로서 사용한다.
```javascript

  class Book {

    // Static Field
    static author = 'Jay Ian';

    // 생성자Contructor
    constructor(title) {
      // Initial field
      this.title = title;
    }

    // Static Method
    static getAuthor() {
      console.log(Book.author);
      return Book.author;
    }
    
  }
  
  const book1 = new Book('Basic of JS');
  const book2 = new Book('Basic of Java');

  console.log(book1.author);
  // [Result] undefined
  console.log(Book.author);
  // [Result] Jay Ian
  
  Book.getAuthor();
  // [Result] Jay Ian

```

##### 5) 상속 Inheritance
&nbsp;-&nbsp; **`extends`** 키워드를 사용하여 객체를 상속 받을 수 있다.
```javascript

  class Book {

    // 생성자Contructor
    constructor(title, author) {
      // Initial field
      this.title = title;
      this.author = author;
    }

    showBookInfo() {
      console.log(`Title : ${this.title}, Author : ${this.author}`)
    }

  }

  // Book 클래스를 상속
  class Novel extends Book {
  }

  // Book 클래스를 상속 후 Book 클래스의 메소드 Override
  class Magazine extends Book {

    // Override
    showBookInfo() {
      super.showBookInfo(); // 부모 객체의 메소드 호출
      console.log(`Title : ${this.title}, Author : ${this.author} Magazion!!!`);
    }
  }
  
  const novel= new Novel('Clean Code', 'Jay Ian');
  const magazine = new Magazine('Top Programming Language');
  console.log(novel.showBookInfo());    
  // [Result] Title : Clean Code, Author : Jay Ian
  console.log(magazine.showBookInfo());
  // [Result]  
  //    Title : Top Programming Language, Author : undefined
  //    Title : Top Programming Language, Author : undefined Magazion!!!

```
<br />

##### 6) instanceof
&nbsp;-&nbsp; **`instanceof`** 키워드를 이용하여 어느 객체를 상속받은 여부를 알 수 있다.
```javascript

  class Book {

    // 생성자Contructor
    constructor(title, author) {
      // Initial field
      this.title = title;
      this.author = author;
    }

    showBookInfo() {
      console.log(`Title : ${this.title}, Author : ${this.author}`)
    }

  }

  // Book 클래스를 상속
  class Novel extends Book {
  }

  // Book 클래스를 상속 후 Book 클래스의 메소드 Override
  class Magazine extends Book {

    // Override
    showBookInfo() {
      super.showBookInfo(); // 부모 객체의 메소드 호출
      console.log(`Title : ${this.title}, Author : ${this.author} Magazion!!!`);
    }
    
  }
  
  const novel= new Novel('Clean Code', 'Jay Ian');
  const magazine = new Magazine('Top Programming Language');

  console.log(novel instanceof Object);         // [Result] True
  console.log(novel instanceof Book);           // [Result] True
  console.log(novel instanceof Novel);          // [Result] True
  console.log(novel instanceof Magazine);       // [Result] False
  console.log(magazine instanceof Object);      // [Result] True
  console.log(magazine instanceof Book);        // [Result] True
  console.log(magazine instanceof Novel);       // [Result] False
  console.log(magazine instanceof Magazine);    // [Result] True

```
