[VsCode] GitGraph Plugin 사용하기

Visual Studio Code(VsCode) 의 extension "git graph" 설치하고 설정하는 법

{
  "author": "Onigiri.J",
  "createdAt": "2024-05-29",
  "updatedAt": "2024-05-29"
}

Remark​

  • GitGraph 는 Git을 GUI 로 git 을 다룰수 있는 툴이다.

설치​

  • VsCode extension 검색에서 찾아 설치할수 있다.
    • MarketPlace - Git Graph

설정 및 실행​

  • Command Palette (ctrl(command) + shift + p) 에서 Git Graph: View Git Graph(git log) 을 실행하면 바로 사용할 수 있다.
    • 이 외에 VsCode 왼쪽 아래의 Git Graph 를 클릭하거나 VsCode의 Souce Control (ctrl + shift + g) 의 Repository 에 있는 View Git Graph 버튼을 클릭한다.

Git Graph 가 Git Repository 를 찾지 못할 때​

  • 간혹 Git Graph 를 설치하고 View Git Graph 실행했을 때, Git Repo 를 찾지 못했다는 화면이 나오는 경우가 있다.

    • Re-Scan 해도 못찾는 경우가 있다.

  • VsCode 의 Workspace 에 Git Repo(.git 폴더) 가 없기 때문이다.

    • Workspace 디렉토리가 아니라 하위 디렉토리에 있을 경우에도 찾지 못한다.

Solution​

  • 아래의 2가지 중 하나의 방법으로 해결 할 수 있다.
      1. Git Repo 가 있는 디렉토리를 Workspace 로 열어준다.
      • Workspace 의 Root 위치에 Git Repo 가 있으면 된다.
      1. Git Graph 의 설정값을 변경한다.
      • VsCode 설정(Preferences -> Settings)이나 extension 설정()@ext:mhutchie.git-graph) 을 연다.
      • @ext:mhutchie.git-graph max depth 를 검색한다.
      • 값을 0 에서 다른 숫자로 변경한다.
        • Root 위치가 0 이므로 하위 디렉토리의 깊이 만큼 숫자를 변경하면 Git graph 가 Repo 를 찾을 수 있다.
